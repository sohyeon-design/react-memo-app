# 📝 메모 앱

React로 만든 간단하고 직관적인 메모 앱입니다.

## 주요 기능

### 1. 새 메모 추가
- **➕ 새 메모** 버튼을 클릭하면 새로운 메모지가 생성됩니다.
- 자동으로 수정 모드로 전환되어 바로 입력할 수 있습니다.

### 2. 메모 수정
- **✏️ 수정** 버튼을 클릭하면 메모를 수정할 수 있습니다.
- 텍스트 영역이 활성화되어 내용을 편집할 수 있습니다.

### 3. 메모 저장
- 수정 모드에서만 **💾 저장** 버튼이 표시됩니다.
- 저장을 클릭하면 변경사항이 저장되고 수정 모드가 종료됩니다.
- 메모 내용이 비어있으면 저장할 수 없습니다.

### 4. 메모 삭제
- **🗑️ 삭제** 버튼을 클릭하면 메모를 삭제할 수 있습니다.
- 삭제 전 확인 메시지가 표시됩니다.

### 5. 메모 검색
- 상단의 검색창에서 메모를 검색할 수 있습니다.
- 검색어가 포함된 메모만 실시간으로 필터링되어 표시됩니다.
- **✖️** 버튼을 클릭하면 검색어를 지울 수 있습니다.

## 추가 기능

- **로컬 스토리지 저장**: 브라우저를 닫아도 메모가 유지됩니다.
- **생성 시간 표시**: 각 메모의 생성 시간이 표시됩니다.
- **반응형 디자인**: 모바일과 데스크톱 환경에서 모두 사용 가능합니다.

## 설치 및 실행

```bash
# 개발 서버 실행
npm start

# 빌드
npm run build
```

## 사용 기술

- React (Create React App)
- CSS3
- Local Storage API
